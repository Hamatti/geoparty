What is this about?

This little Jeopardy clone was made during the Chartio Hackathon (Oct 16th-17th, 2014) for fun and no other purpose.

I found a JSON full of questions from real Jeopardy TV shows some time ago and when I found out Watson API was published, I knew I wanted to hack something around the topic. The game runs on Node.js, is hacky, probably breaks a lot and doesn't have all the features yet. Basic multiplayer game works, Watson API not yet.
